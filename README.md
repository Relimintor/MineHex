# MineHex

MineHex is a tiny Minecraft-inspired sandbox prototype where the world is made of **hexagonal tiles** instead of square blocks.

## Features

- Procedurally generated hex world with several materials.
- Place terrain by clicking any hex tile.
- Break terrain with `Shift + Click`.
- Move a player token with `W/A/S/D`.
- One-file frontend with no build tooling.

## Run

Open `index.html` in your browser.

If you want to serve it locally:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
